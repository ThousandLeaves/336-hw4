Strings

The string in Javascript is a built-in object. You can create a string by assigning it with either the double " " or single ' ' quotation marks. It's important to note that Javascript does not have a char type for single characters. In addition to the methods above, a string can also be declared with a backtick. ` ` This allows us to execute an expression similarly to as if you were using Linux shell scripting.

let myString1 = "This is a string."

let myString2 = 'This is also a string.'

let myString3 = `With backticks, we can embed expressions like this ${Math.pow(2,3)}. Neat!`

String methods

Usage: let myChar = myString.charAt(i)

The index at i refers to the specific character in the string, starting at 0. Indexes cannot be negative. Instead use myString.length - 1

Usage: matchString = myString.match(regex)

The match method will check a string's contents against a regular expression and return the first result or all results, the latter only being true if the global flag 'g' is set in the expression. The matter of regular expressions is complex and the match options are diverse. You can find out more about them on this external website.

Usage: newString = myString.substring(i, j)

Substring creates a new string based on a fragment of an existing string. The start and end points of i and j refer to the start and end points of the region of the original string to capture. If no end point is specified, substring will return the remaining portion of that string. As before, the first character of a string begins at index 0. When using the substring method, be aware that it does not change the original string. You will need to assign the results of the operation to that original string if you wish for it to become permanent.

Note: Use of the similarly named String.substr method is discouraged. It is older and remains for legacy purposes.

This simple function will change the type of a variable to string. It is not always used, as this operation occurs implicitly when concatenating strings and numbers with the '+' operator.
