Arrays

As in most programming languages, the array in Javascript is a built-in object. While Javascript allows you to store key-value pair objects, arrays work like an ordered-list that we can re-order and iterate through. Unlike some languages, the size of an array does not need to be set on declaration, and arrays are effectively dynamic. All arrays are capable of accepting mixed data types.

Array methods

Usage1: myArray.forEach(callback(elem1, [elem2, ...]))

Usage2: myArray.forEach((parameters) => { statements });

The forEach method is a syntactically simpler method of performing an iterative loop over all array elements. It will operate on an array, running a specified method on each element. The closest traditional comparison to this operation is the for loop.

In the first usage example, a callback function is supplied. The second use case shows an arrow function expression instead.

Usage: let remElement = myArray.pop()

Using the pop method will remove the last element from an array. It is important to note that pop() will change the original array. The returned value is the element that was on the end of the array.

Usage: let remElement = myArray.shift()

Using the shift method will remove the first element from the front of an array. It is important to note that shift() will change the original array. The returned value is the element that was originally the first element in the array.

Note: The similarly named method "Array.unshift()" will add an element to the beginning of an array instead.

Usage: let slicedArray = slicedArray.slice([start[, end])

The slice method will remove a portion of an array based on the specified start point and up to (but not including) the given end point. Define these points like you would the indices of the array. If the start point is given as 1 and the end point is 4, the slice function would remove the 2nd element up to, but not including the 5th element. You may also omit an end point and slice out all of the array following the start index.

It is important to note that with the slice method, the original array is not modified. You must assign the operation to the original array if you wish for the change to be permanent.
